## 👩‍💻 Installation
Follow the commands below to clone this repository on your system

```bash

1. git clone https://github.com/codeshaan/whatsapp_clone.git

2. cd whatsapp_clone

3. npm install

4. npm start

```

  

## ⏬ Tech Used

* ReactJS 🔥

* Firebase

* Material UI

* CSS

  
  

## 👇 Features

* ⚒ Authentication (Google OAuth)

* 🗯Real time chat

* 🌈 Can create new rooms for chat
